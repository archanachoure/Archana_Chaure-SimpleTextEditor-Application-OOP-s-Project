# 📝 Text Editor

A simple yet functional **desktop text editor** built with Java Swing. It supports basic file operations and text editing — clean, lightweight, and easy to run.

---

## 🖥️ Features

- **New File** — Clear the editor and start fresh
- **Open File** — Browse and open any `.txt` or plain text file
- **Save File** — Save to an existing file or choose a new location
- **Edit Menu** — Cut, Copy, Paste, and Select All support
- **Monospaced Font** — Clean `Monospaced 14pt` font for comfortable reading
- **Scroll Support** — Handles large files with a scrollable text area

---

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or higher installed
- A terminal / command prompt

### Run the App

```bash
# Clone the repository
git clone https://github.com/archanachaure/text-editor.git
cd text-editor

# Compile
javac TextEditor.java

# Run
java TextEditor
```
